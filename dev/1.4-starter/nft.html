Items Of Note:

  • HTML data attributes can be leveraged to provide stateful information
  • <script> location matters
  • Checking for initial content being loaded is important for proper performance
  • innerHTML sets a property, not a node
  • appendChild attaches a node within another element
  • Object literals allow for data cleaning
  • String literals can be used to build HTML templates
  • Tools and packages like faker are extremely helpful in preparing an interface for real data
  • Older syntax can still be very useful!
