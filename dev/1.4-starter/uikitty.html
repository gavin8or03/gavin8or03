API

Items Of Note:

  • Some APIs require a Key, token, or both for access
  • Protecting your Keys is very important
  • Check for errors when using APIs
  • Promises was be resolved then converted into a usable format such as JSON
  • Destructuring API properties is very helpful
  • Opening in a new tab or window requires noopener and noreferrer to be set
  • Working with an API's data is just like local data, we just need to wait for it!
