Smush The Tabs

Items Of Note:

  • There are no tab or tablist elements native to HTML
  • Tabs present an opportunity to learn about ARIA roles
    • Try using VoiceOver or Narrator to see this in action
  • Keyboard left and right through the tabs
  • Use Enter or Space to select tabs
  • Use Tab to move between sections of the site
  • An image's alt text must be set for a screen reader to be able to describe it.
  • alt text also helps with broken images!
  • It is best practice to use buttons for on page interactions that do not link to another page.
